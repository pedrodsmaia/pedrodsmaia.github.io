# Personal Website

This is a simple personal website that can be hosted for free with GitHub Pages.

## Edit the content

Open `/Users/pedro.dossantosmaia/Documents/Playground 4/index.html` and replace:

- `Your Name`
- the short bio text
- the project descriptions
- the contact links

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload these files:
   - `index.html`
   - `styles.css`
   - `script.js`
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and the `/ (root)` folder.
6. Save.

GitHub will publish your site at:

`https://yourusername.github.io/repository-name/`

If the repository is named `yourusername.github.io`, your site will be:

`https://yourusername.github.io/`
